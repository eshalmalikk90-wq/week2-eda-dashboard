# 📊 Student Performance EDA Dashboard (Streamlit)

## 📌 Project Overview
This project is an interactive Exploratory Data Analysis (EDA) dashboard built using Streamlit. It analyzes a student performance dataset to uncover insights, patterns, and relationships between different academic and demographic features.

The goal of this project is to practice real-world data cleaning, preprocessing, visualization, and dashboard development using Python.

---

## 📂 Dataset Description
The dataset contains student performance records with 1000+ entries. It includes features such as:

- Gender
- Parental level of education
- Lunch type
- Test preparation course
- Math score
- Reading score
- Writing score

---

## 🧹 Data Cleaning Process
The following data cleaning steps were performed:

- Handling missing values
- Removing duplicate records
- Checking and correcting data types
- Outlier detection using boxplots
- Ensuring data consistency

---

## 📊 Exploratory Data Analysis (EDA)

The following analyses were performed:

### ✔ Statistical Summary
Basic descriptive statistics (mean, median, standard deviation) were calculated.

### ✔ Distribution Analysis
Histograms were used to understand how student scores are distributed.

### ✔ Correlation Analysis
Relationships between math, reading, and writing scores were analyzed using a heatmap.

### ✔ Outlier Detection
Boxplots were used to identify extreme values in the dataset.

---

## 📈 Features of the Streamlit Dashboard

- Dataset overview (shape, columns, data types)
- Missing value analysis
- Duplicate record detection
- Interactive filters (sidebar)
- Dynamic histograms
- Boxplot visualizations
- Correlation heatmap
- User-controlled visualizations
